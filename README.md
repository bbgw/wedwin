
# 使用说明
1. 设置密码阶段可以随时退出。
1. 通过更改代码文件`main.py`中第行`total`的值可以更改密码长度。
1. 密码按位设置；每位密码只允许输入一个字符，按回车键录入。
1. 一旦密码设置完成，无法自行退出；强制退出方式会导致密码丢失。

# 破解说明
1. 密码保存在名为`pwd`的`str`类型变量中。
1. 请将猜测密码保存在`guess`变量中；`guess`已初始化为空字符`''`。
1. 系统会自动比对`guess`和`pwd`，若匹配则通关成功。
